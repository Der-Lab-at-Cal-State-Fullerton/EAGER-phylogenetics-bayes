# EAGER-phylogenetics-bayes
This is a basic tutorial for running a phylogenetic analysis in MrBayes. The tutorial was first developed as a class assignment by Emma Gomez for the EAGER Program at Cal State Fullerton in Spring 2026.

Data used in this tutorial have been modified from
Krishnankutty, Sindhu; Dietrich, Christopher; Dai, Wu; Siddappaji, Madhura (2018): NEXUS data file for phylogenetic analysis of Iassinae (Hemiptera: Cicadellidae). University of Illinois at Urbana-Champaign. https://doi.org/10.13012/B2IDB-9500981_V1 

For tutorial instructions, please see the [MrBayes Instructions](./mb_instructions.md)
